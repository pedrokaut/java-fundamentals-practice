# Java Fundamentals Practice

Este repositório contém exercícios desenvolvidos em Java com foco no fortalecimento da lógica de programação e nos fundamentos da linguagem para.

##Conteúdos abordados
- Estruturas condicionais
- Estruturas de repetição
- Métodos
- Arrays
- Conceitos básicos de Programação Orientada a Objetos (quando aplicável)

##Objetivo
Consolidar conhecimentos em Java e aprimorar a base necessária para o desenvolvimento de aplicações mais complexas.
